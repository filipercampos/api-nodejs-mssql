# Description
API NodeJS MSSQL open api 3.0.0

## Running the app
```bash
# start app
$ npm start

# dev mode (nodemon)
$ npm run dev

# debug mode
$ npm run debug

# start with pm2
$ npm run pm2
```
## Test

```bash

# test end to end
$ npm run test:e2e

# unit tests
$ npm run test:u

# individual tests or other
$ npm run test:i

# all tests
$ npm test
```

# packages
Update dependencies
```bash

# install package
$ npm i -g npm-check-updates

# update package.json
$ ncu -u

# install update
$ npm install
```
